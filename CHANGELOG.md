## [0.0.1] - 2025-06-15

### Added
- **Initial Release**: First version of the KvS-2077 theme.
- **Theme File**: Core theme definition `themes/KvS-2077-color-theme.json` with a complete color palette for UI and syntax highlighting.
- **Extension Manifest**: `package.json` containing the theme's metadata and contribution points.
- **Documentation**: Initial `README.md`, `CHANGELOG.md`, and a quick start guide.
- **Configuration**: Project configuration files including `.vscode/launch.json`, `.vscodeignore`, `.gitignore`, and `.gitattributes`.
